# AI-Native Textbook Builder Constitution

## Core Principles

### Content-first
Textbook chapters must be well-written, deep, and pedagogical.

### Highly structured
Chapters → Sections → Subsections → Examples → Diagrams.

### MDX-friendly
Equations, code blocks, diagrams, images.

### Frontend
Responsive, modern, clean, and academic.

### AI chat assistant
Included, specialized in Physical AI.

### Navigation and Features
Must support table of contents, search, glossary, footnotes.

### Tone
University-level but approachable.

### Scientific accuracy
Maintain scientific accuracy on robotics: ROS 2, Gazebo, Isaac, VLA.

## Governance
Constitution supersedes all other practices; Amendments require documentation, approval, migration plan. All PRs/reviews must verify compliance; Complexity must be justified; Use CLAUDE.md for runtime development guidance.
